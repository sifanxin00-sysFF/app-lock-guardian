# 守门监管 UI 参考板

## 1. 最终融合风格

守门监管不是儿童软件，也不是后台管理页。最终风格定义为：

**家庭守护型原生 App。**

视觉关键词：

- 暖白：页面接近白色，只带一点家庭温度，不黄腻。
- 轻卡片：卡片只用于承载任务和控制，不用一层套一层。
- 精致图标：图标像 App 组件，不像临时线框。
- 真实插画：家庭、手机、头像要有真实视觉重量，不再像 CSS 拼图。
- 原生开关：设备控制像手机系统设置，不像网页按钮。
- 少装饰：装饰必须服务状态和情绪，不能抢内容。
- 少粗字：标题稳，不吼；正文轻，信息层级清楚。
- 少边框：靠留白、字体、颜色层级分组，不靠一堆框。

一句话目标：

**Google Family Link 的清爽结构 + Bark/Qustodio 的家庭安全感 + AirDroid 的设备控制组织 + Apple/Material 的原生控件质感。**

## 2. 首页参考

主参考：Google Family Link  
辅助参考：Bark、原目标视觉稿

学什么：

- 首页只呈现最重要的监管状态，不做后台 dashboard。
- 顶部信息要轻：标题、刷新、设置，不做厚工具栏。
- 状态数字要清楚，但不能像统计后台。
- 待审批申请应该像一张当前任务卡，而不是一块表单。
- 家庭场景插画要提供安心感，但不能挡住内容。

不学什么：

- 不做大量功能入口。
- 不做当前设备卡回到首页。
- 不做过粗标题、过厚卡片、过重阴影。
- 不做满屏黄奶油色。

首页 v2 目标样子：

- 顶部「守门监管」为稳重品牌标题，字号比上一版小一档。
- 右上刷新和设置是轻量文字图标按钮，无白色圆底。
- 横幅是浅色家庭风景，像插画资产，不像 CSS 色块。
- 两个统计块横向排列：待审批、正在放行。
- 待审批卡一张，按钮少而明确。

## 3. 设备页参考

主参考：AirDroid Parental Control  
辅助参考：Apple 设置页、Material Switch

学什么：

- 设备页是工具页，重点是“当前设备”和“控制状态”。
- 一个页面只控制当前选中的一个设备。
- 开关必须像原生 App 控件，能一眼看出开/关。
- 命令状态是辅助信息，不能像日志后台。
- 设备图要真实，帮助用户确认当前控制对象。

不学什么：

- 不堆多个设备控制台。
- 不恢复 DeepSeek、守门模式、命令状态 tab。
- 不用两个大按钮假装开关。
- 不把命令列表做成后台日志。

设备页 v2 目标样子：

- 顶部居中标题「设备控制」，右侧状态入口轻量。
- 设备卡左侧是真实感手机图，右侧是设备名、在线、最近操控时间、型号。
- 后台保护、高风险入口、桌面图标都是开关卡。
- 「立刻重新锁紧」作为辅助动作，不抢后台保护开关。
- 最近远程命令只显示 2-3 条轻量记录。

## 4. 我的页参考

主参考：Google Family Link 账号/家庭设置  
辅助参考：Apple 设置页

学什么：

- 我的页是账号和记录页，不是控制台。
- 账号卡要有温馨头像或家庭插画。
- 审批记录像 App 列表项，左图标、中内容、右状态。
- 修改密码、刷新请求、退出登录像设置项，位置靠后。
- 添加桌面提示要轻，像帮助说明，不像错误提示。

不学什么：

- 不恢复命令状态 tab。
- 不把审批记录做成表格。
- 不把退出登录做成特别醒目的危险区。
- 不写虚假的一键安装。

我的页 v2 目标样子：

- 顶部标题「我的」稳重居中。
- 账号卡更白更轻，头像更精致。
- 审批记录只展示最近几条，列表行距舒服。
- 设置项统一白底，带轻箭头。
- 添加桌面引导放在底部，文案简短。

## 5. 控件参考

底部 Tab：

- 参考 Apple / Material 的三项底部导航。
- 当前项用橙色强调，其他项灰黑。
- 图标和文字垂直对齐，不能被安全区裁切。
- Tab 不能遮挡页面内容，页面底部必须预留足够空间。

开关：

- 开关必须是真实开关形态，不是按钮。
- 开启：青绿色轨道 + 白色圆点靠右。
- 关闭：浅灰轨道 + 白色圆点靠左。
- 开关旁必须有标题和一句短说明。

列表：

- 左图标，中间主次信息，右状态或箭头。
- 有箭头就必须可点击；不可点击就去掉箭头。
- 状态标签不能压住图标和标题。

按钮：

- 主按钮只给审批、保存、查看方法这类明确动作。
- 危险按钮只给拒绝、重新锁紧、退出登录。
- 按钮不贴边、不互相挤压。

## 6. v2 验收线

v2 静态稿必须比上一版明显提升：

- 首页不再像大字海报。
- 设备页不再像功能堆叠页。
- 我的页不再像简单列表页。
- 手机图、头像、应用图标不能像占位。
- 底部 Tab 不遮挡内容。
- 每页单项评分不低于 8.5。
